# Catalógo de produtos
*Projeto construído com bootcamp, a aplicação é baseada e microsserviços utilizando spring cloud.*

## Tecnologias usadas
- String Framework
  - Spring Web
  - Spring Cloud
  - Spring ElasticSearch
  - Spring actuator
  - Spring cloud netflix-eureka-client
  - Gradle
  - Java
